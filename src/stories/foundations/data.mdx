# Data

Because color affects how viewers perceive and prioritize information, careful attention to color choices is essential for clear and accurate data visualizations.

## Categorical

Apply categorical colors whenever you need to visually distinguish between two or more separate, unrelated data groups.

## Monochromatic

Use a monochromatic color scale when representing ranges of data within the same category. In this approach, the darkest shade should indicate the highest or largest value, while lighter shades correspond to smaller values.

## Status

These data visualization colors rely on familiar color connections to represent status and urgency. When there are multiple levels within the same status category, use a bold color variant to highlight the more critical or intense value.
